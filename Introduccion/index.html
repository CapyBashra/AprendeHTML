Introducción a HTML

¿Qué es HTML?

HTML, siglas en inglés de HyperText Markup Language (‘lenguaje de marcado de hipertexto’), es un lenguaje de marcado utilizado para elaborar y estructurar el contenido de las páginas web. No es un lenguaje de programación, sino más bien un conjunto de etiquetas y elementos que le indican al navegador cómo mostrar y organizar el contenido en una página web; como texto, imágenes, videos, juegos, entre otros.

Elementos

Partes de un elemento HTML

El elemento consta de tres partes principales: la etiqueta de apertura, el contenido y la etiqueta de cierre.

  • Etiqueta de apertura: Es la parte inicial del elemento y se representa mediante el nombre del elemento envuelto entre símbolos de menor que ("<") y mayor que (">"). Por ejemplo, <p> es la etiqueta de apertura de un párrafo.
  • Contenido: Es el texto o información dentro del elemento. Representa el contenido visible que se mostrará en la página web.
  • Etiqueta de cierre: Es similar a la etiqueta de apertura, pero con una barra diagonal ("/") antes del nombre del elemento, indica la parte final del elemento. Por ejemplo, </p> es la etiqueta de cierre de un párrafo.

Es importante incluir tanto la etiqueta de apertura como la de cierre para evitar errores y resultados inesperados.

Elementos anidados

En HTML, es posible colocar elementos dentro de otros elementos, lo cual se conoce como anidamiento. Esto permite estructurar y organizar el contenido de una página web de manera jerárquica y lógica.

Al anidar elementos, uno se convierte en el elemento padre y el otro en el elemento hijo. El elemento hijo está contenido dentro del elemento padre, lo que significa que está encapsulado por él. Esto se logra al colocar el elemento hijo entre la etiqueta de apertura y la etiqueta de cierre del elemento padre.

Por ejemplo, considera el siguiente código HTML:

<p>Este es un párrafo <strong>importante</strong> en negrita.</p>

En este caso, el elemento <p> es el elemento padre y contiene al elemento <strong>, que es el elemento hijo. El resultado visual en un navegador web sería: "Este es un párrafo importante en negrita." El texto "importante" se muestra en negrita debido al uso del elemento <strong> dentro del elemento <p>.

Los elementos anidados pueden ser utilizados para aplicar estilos específicos, crear estructuras complejas y definir relaciones entre diferentes partes del contenido. Es importante tener en cuenta que los elementos anidados deben abrirse y cerrarse en el orden correcto para mantener una estructura válida.

Por ejemplo, el siguiente código sería inválido debido a un anidamiento incorrecto:

<p>Este es un párrafo <strong>importante en negrita.</p></strong>

Elementos vacíos

No todos los elementos siguen el patrón de etiqueta de apertura, contenido y etiqueta de cierre.
Los elementos vacíos, también conocidos como elementos nulos o vanos, son elementos HTML que no tienen contenido ni etiqueta de cierre. Estos elementos consisten en una etiqueta única, que se utiliza para insertar o incrustar algo en una página web, como imágenes, saltos de línea u otros tipos de contenido que no requieren contenido adicional o etiqueta de cierre. Por ejemplo, para insertar una imagen dentro de una pagina web se usa el elemento <img> de la siguiente manera:

<img src="https://upload.wikimedia.org/wikipedia/commons/3/35/Finchen.jpg" alt="Imagen de un gato doméstico llamado Finchen"/>

Atributos

Un atributo está compuesto por:

  • Un espacio entre este y el nombre del elemento. (Para un elemento con más de un atributo, los atributos también deben estar separados por espacios).
  • El nombre del atributo, seguido por un signo igual. El nombre proporciona información sobre qué propiedad se está modificando. Algunos ejemplos comunes de nombres de atributos incluyen src, href, class, id, alt, width, height, entre otros.
  • Un valor del atributo, rodeado de comillas de apertura y cierre. Es el valor asignado al atributo y define la configuración o el contenido específico para ese atributo. El valor puede ser una cadena de texto, un número, un color, una URL u otro tipo de dato dependiendo del atributo.

Atributos booleanos

A diferencia de los atributos regulares que tienen un nombre y un valor, los atributos booleanos se escriben simplemente como nombres de atributos sin un valor. Estos atributos se utilizan para indicar la presencia o ausencia de una característica o propiedad particular en un elemento HTML. La presencia del atributo por sí sola indica que la característica está activada, mientras que la ausencia del atributo implica que la característica está desactivada.

Por ejemplo, el atributo booleano disabled indica que un elemento interactivo está desactivado o inhabilitado, así:

<input type="text" disabled />
<input type="text" />
